# Odin-Etch-A-Sketch

A browser-based interactive drawing grid inspired by the classic Etch-A-Sketch, built using HTML, CSS, and vanilla JavaScript. Users can dynamically resize the grid, draw with random or custom colors, and reset the canvas at any time.

## How It Works

1. A container is divided into equal-sized squares using Flexbox.
2. JavaScript dynamically generates the grid based on user input.
3. Each square listens for a mouseenter event to apply color.
4. Users can resize the grid, select colors, and reset the board.
